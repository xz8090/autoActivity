# autoActivity
## 北化活动报告监听系统

### 项目说明：
项目基于Java无头浏览器[jBrowserDriver](https://github.com/MachinePublishers/jBrowserDriver)开发的。  
该版本为PC无界面版，可以直接在编译器中运行，或者打包后使用命令运行，注意配置文件地址。  
配置文件包含学号密码邮箱等信息配置，注意字段是否为空。  
不在校园网下需要配置WebVPN的登录密码。  
本项目不提供验证码识别服务，请自己在CodeIdentification.java增加相关验证码识别功能。  
代码仅供学习参考，不得非法滥用。
